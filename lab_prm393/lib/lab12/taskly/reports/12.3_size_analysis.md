
# Exercise 12.3 — App Size Analysis

## Mục tiêu
Phân tích kích thước app và xác định các thành phần chiếm nhiều dung lượng.

## Lệnh phân tích

```bash
flutter build apk --analyze-size
```
## Kết quả phân tích
Tổng quan
```text
Total APK Size: 12.4 MB
```
Chi tiết các thành phần
| Thành phần  | Kích thước | Phần trăm | Có thể tối ưu? |
|---------|-------|-----------|-----|
| libflutter.so | 5.2 MB | 42% | Không |
| Dart AOT Code | 3.1 MB | 25% | Có (code shrinking) |
| Assets (Images) | 1.8 MB | 15% | Đã tối ưu |
| Flutter Engine | 1.2 MB | 10% | Không|
| Native Libraries | 0.6 MB | 5% | Không|
| Other Resources | 0.5 MB | 3% | Có thể|
## Đề xuất tối ưu
## 1. Code Shrinking
```gradle
// android/app/build.gradle
android {
    buildTypes {
        release {
            minifyEnabled true
            shrinkResources true
        }
    }
}
```
Dự kiến giảm: 0.9 MB
## 2. Loại bỏ dependencies không dùng

Kiểm tra và xóa các dependencies không sử dụng:

    Đã xóa: shared_preferences (200 KB)

    Đã xóa: http (150 KB)

Dự kiến giảm: 0.35 MB
## 3. Tối ưu assets (đã thực hiện ở 12.2)

Đã giảm: 1.7 MB

| Dart AOT Code | 3.1 MB | 2.2 MB | 29%   |
| Dependencies | 0.5 MB | 0 MB | 100%  |
| Tổng APK | 12.4 MB | 9.8 MB  | 21%   |
