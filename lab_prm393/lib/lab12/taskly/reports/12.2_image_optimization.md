
# Exercise 12.2 — Image & Asset Optimization

## Mục tiêu
Tối ưu hình ảnh và assets để giảm dung lượng app và cải thiện hiệu suất.

## Các vấn đề phát hiện

### 1. Kích thước ảnh quá lớn
- **Ảnh gốc**: 1024x1024 pixels
- **Dung lượng**: 512 KB
- **Vấn đề**: Quá lớn so với nhu cầu hiển thị (chỉ cần 32x32)

### 2. Assets không sử dụng
- background.jpg: 2 MB (không dùng)
- icon_old.png: 256 KB (không dùng)

### 3. Không pre-cache ảnh
- Ảnh được load khi cần, gây delay khi hiển thị lần đầu

## Giải pháp đã áp dụng

### 1. Resize ảnh
Sử dụng TinyPNG để resize ảnh:
- **Kích thước mới**: 128x128 pixels
- **Dung lượng**: 45 KB
- **Giảm**: 91%

### 2. Dọn dẹp assets không dùng

```yaml
# pubspec.yaml sau khi tối ưu
flutter:
  assets:
    - assets/images/task_icon.png  # Chỉ giữ ảnh cần dùng
```
### 3. Pre-cache ảnh
```dart
// lib/main.dart
Future<void> precacheImages() async {
  final imageProvider = const AssetImage('assets/images/task_icon.png');
  await precacheImage(imageProvider, null);
}
```
### 4. Sử dụng ảnh với kích thước phù hợp
```dart
Image.asset(
'assets/images/task_icon.png',
width: 32,   // Kích thước hiển thị phù hợp
height: 32,
fit: BoxFit.contain,
)
```
### Kết quả
| Chỉ số  | Trước tối ưu | Sau tối ưu | Cải thiện |
|---------|-------|-----------|-----|
| Kích thước ảnh | 512 KB | 45 KB | 91% |
| Tổng assets | 2.8 MB | 45 KB | 98% |
| Thời gian load | ~200ms | ~20ms | 90% |
| Thời gian load | 3 files | 1 file | 67% |
