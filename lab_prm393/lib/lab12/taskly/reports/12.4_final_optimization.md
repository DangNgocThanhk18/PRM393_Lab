## Performance Checklist

### 1. Debug Logs
```dart
// Sử dụng kDebugMode để chỉ in log khi debug
if (kDebugMode) {
  print('Adding task: $title');
}
```
### 2. Const Constructors
```dart
// Tất cả widget đều có const constructor
const TaskTile({super.key, required this.task});
const TaskListScreen({super.key});
```
### 3. Proper Keys
```dart
// Sử dụng ValueKey cho danh sách
ListTile(
key: ValueKey(task.id),
// ...
)
```
### 4. Constants cho styles
```dart
class AppConstants {
  static const double iconSize = 32.0;
  static const EdgeInsets listPadding = EdgeInsets.all(16.0);
}
```
### 5. Pre-cache Images
```dart
Future<void> precacheImages() async {
  final imageProvider = const AssetImage('assets/images/task_icon.png');
  await precacheImage(imageProvider, null);
}
)
```
### 6. Clean Project
```bash
flutter clean
flutter pub get
)
```
### 7. Run Analyzer
```bash
flutter analyze
)
```
### 8. Run Tests
```bash
flutter test
```
